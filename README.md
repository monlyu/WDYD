# WDYD 
你做了什么？
<hr/>
##目的？
每天你都做了很多很多的事情，老板总算要求你把每天做的事情都记录下来，然后每个月都要写一个报告做了什么？

## 技术
Java Servlet + Sqlite

## 如何部署

1. 安装 JDK 
2. 安装 TOMCAT
3. 修改 config.cfg 文件
4. sqlitepath 对应的数据库位置
5. projects=Wyical   //随时增加你需要列举的项目

## MAC APP 地址的修改

修改: WortTime.app/Contents/Info.plist 
URL参数就可以
